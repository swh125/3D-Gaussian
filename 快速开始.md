# 🚀 快速开始 - 使用新照片

## 方法一：最简单（推荐）

1. **打开 `run_with_new_images.sh` 文件**
2. **修改第5行的照片路径**：
   ```bash
   PHOTOS_PATH="你的照片文件夹路径"  # 例如: "C:/Users/hp/Desktop/my_photos"
   ```
3. **运行脚本**：
   ```bash
   bash run_with_new_images.sh
   ```

## 方法二：直接提供路径给我

**直接把照片路径发给我，我会帮你修改脚本并运行！**

例如：
- 照片文件夹路径：`C:\Users\hp\Desktop\my_photos`
- 或者视频文件路径：`C:\Users\hp\Desktop\my_video.mp4`

## 注意事项

1. **照片格式**：支持常见图片格式（jpg, png等）
2. **照片数量**：建议至少20-50张不同角度的照片
3. **照片质量**：照片之间需要有重叠区域（用于3D重建）
4. **输出目录**：脚本会自动创建输出目录，无需手动创建

## 运行后

脚本会自动完成：
1. ✅ 数据处理（COLMAP重建）
2. ✅ 3D Gaussian Splatting训练
3. ✅ SAM掩码提取
4. ✅ 对比特征训练

完成后，运行GUI进行交互式分割：
```bash
python saga_gui.py --model_path <模型路径> --data_path <数据路径>
```
